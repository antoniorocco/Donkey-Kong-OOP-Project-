Donkey Kong (OOP Project with Allegro) 

[![Watch the video](https://i.imgur.com/ST0QkxC.png)](https://www.youtube.com/watch?v=RMV6-3SdDWg&t=11s)


Tested on Linux (Ubuntu 19.10) / macOS (majove)

# Dep

* Make
* Java 8+
* Gradle

```
make dep-install
```

# Run

Game
```
make run
```


Editor
```
make run-editor
```

|key|Action|
|---|------|
|1 | Floor |
|2 | Stair |
|3 | Barrel |
|r | Reset |
|s | Save BonusLevel.map|


Format

```
make format
```
